Technology Used


The technique used in providing the solution i.e. classifying the disease in a crop is image processing with Artificial Intelligence. More precisely, our proposed solution includes a Deep Learning algorithm(Convolutional Neural Network) and performing other mathematical calculations on feed-forward network.


While the crop recommendation tasks will be done by a self-designed algorithm which recommends best crop from the database according to the given conditions entered by the user. The self-designed algorithem consists of a metric system which is capable of calculating similarity score between user input and the crops in the database with additional weightage assigned to features for best prediction.
